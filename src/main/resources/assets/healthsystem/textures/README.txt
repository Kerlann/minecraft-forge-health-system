Placez ici les fichiers de texture pour les icônes et items :

1. Créez un dossier 'gui' pour les éléments d'interface graphique :
   - health_icons.png (64x64) pour les icônes d'état de santé
     Rangée 1 : Saignement (léger), Fracture (légère), Poison (léger), État critique
     Rangée 2 : Saignement (modéré), Fracture (modérée), Poison (modéré), État critique (clignotant)
     Rangée 3 : Saignement (grave), Fracture (grave), Poison (grave), [Réservé]
     Rangée 4 : [Réservé], [Réservé], [Réservé], [Réservé]

2. Créez un dossier 'items' pour les textures des items médicaux :
   - bandage.png (16x16)
   - splint.png (16x16)
   - antidote.png (16x16)
   - first_aid_kit.png (16x16)
   - adrenaline.png (16x16)

Pour les modèles d'items, utilisez le format standard de Minecraft et placez-les dans :
src/main/resources/assets/healthsystem/models/item/
